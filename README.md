# Firebase Studio

This is a NextJS starter in Firebase Studio.

To get started, take a look at src/app/page.tsx.
# install deps
npm install

# start Next dev server (port 9002 per package.json)
npm run dev

# in a second terminal: start GenKit flows (reads src/ai/dev.ts)
npm run genkit:dev

# or run the watcher for GenKit
npm run genkit:watch
